Quem somos nós?

Nós somos um grupo de estudantes universitários e estamos envolvidos em um projeto acadêmico. Estamos criando um site que utiliza tecnologias como HTML, CSS e JavaScript, com a biblioteca Three.js.

Por que este projeto foi criado?

Este projeto foi criado como parte de nosso curso universitário para aplicar e aprimorar nossas habilidades em programação, design web e tecnologias 3D, como o Three.js. Além disso, queremos contribuir de forma positiva, fornecendo um recurso online que permite aos usuários explorar e aprender sobre os pontos turísticos no estado de São Paulo de uma maneira interativa e envolvente. É uma oportunidade de combinar nossa paixão por tecnologia com a promoção do turismo e do patrimônio cultural da região.

Como posso contribuir?

É importante destacar que a ideia de tornar o projeto open-source é uma consideração futura e ainda não está implementada. No momento, o projeto não está disponível como um repositório open-source no GitHub. Portanto, a capacidade de contribuir ainda não está ativa, mas estamos considerando essa possibilidade para o futuro. Estamos entusiasmados com a ideia de abrir o projeto para contribuições externas, mas ainda não chegamos a esse estágio. Fique atento para futuras atualizações e anúncios quando decidirmos avançar nessa direção.
